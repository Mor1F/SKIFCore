**ЗАКЛЮЧЕНИЕ**

В ходе выполнения данной исследовательской работы было спроектировано и успешно реализовано полнофункциональное веб-приложение «SKIF Cloud», представляющее собой защищённое облачное хранилище данных с клиентским шифрованием. Основной целью работы являлось проектирование, программная реализация и тестирование комплексного веб-приложения с использованием библиотеки React.js, интегрированного с защищённой серверной частью и обеспечивающего полный цикл управления файловыми объектами в безопасной среде. Поставленная цель достигнута в полном объёме.

В соответствии с поставленными задачами были получены следующие результаты.

По первой задаче — проведён теоретический анализ архитектурных принципов библиотеки React.js. Изучены механизм Virtual DOM, компонентный подход, функциональные компоненты и хуки. Установлено, что React.js является оптимальным инструментом для разработки высокопроизводительных SPA-приложений благодаря декларативному подходу и развитой экосистеме.

По второй задаче — проведён анализ существующих решений в области облачных хранилищ и выявлен их ключевой недостаток: хранение ключей шифрования на сервере. Спроектирована архитектура системы «клиент-сервер» на основе принципа Zero-Knowledge, при которой сервер работает исключительно с зашифрованными данными и не имеет доступа к их содержимому.

По третьей задаче — разработан и реализован интерактивный пользовательский интерфейс, включающий динамическую панель управления Dashboard с поддержкой загрузки файлов через Drag-and-Drop, систему фильтрации и поиска, адаптивную вёрстку для мобильных устройств и управление корзиной с механизмом мягкого удаления.

По четвёртой задаче — реализованы механизмы управления глобальным состоянием авторизации с использованием React Context API. Обеспечен защищённый обмен данными через JWT-токены с автоматической подстановкой заголовков авторизации посредством перехватчиков Axios.

По пятой задаче — реализована система клиентского шифрования файлов на основе алгоритма AES-256-GCM с использованием браузерного Web Crypto API. Разработан модуль чанковой обработки файлов по 4 МБ, исключающий переполнение оперативной памяти при работе с большими файлами. Реализован модуль keyVault, обеспечивающий персистентное хранение ключей шифрования: ключи защищаются мастер-ключом, деривированным из пароля пользователя через алгоритм PBKDF2, и синхронизируются между устройствами через сервер в зашифрованном виде.

По шестой задаче — проведено функциональное тестирование всех модулей системы. Результаты положительных и негативных сценариев подтвердили корректную работу механизмов авторизации, шифрования, управления файлами и защиты от автоматизированных атак. Все тесты пройдены успешно.

Практическая ценность работы подтверждается тем, что приложение «SKIF Cloud» развёрнуто на реальном сервере и готово к эксплуатации в качестве частной защищённой платформы обмена данными, независимой от сторонних облачных провайдеров.

Среди направлений дальнейшего развития проекта можно выделить следующие: внедрение системы JWT refresh-токенов для повышения безопасности сессий; реализация механизма перешифровки vault при смене пароля пользователя; разработка мобильного приложения на базе React Native; добавление поддержки совместного доступа к папкам с разграничением прав.

---

Готово — все шесть задач закрыты, цель подтверждена, перспективы указаны.
